Site name

The site name is Weather on Point and the domain would be weatheronpoint.com. As the name indicates this will be a website where users will get information about the weather.

Purpose Statement

The purpose of this website would be to help users to know the weather for the day and even the week. This will be fulfilled by displaying the temperatures each day, indicating the percentage of probability of rain and snow.
